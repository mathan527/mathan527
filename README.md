## InsurNova — The Flagship

> **The problem:** Delivery riders on Swiggy, Zomato, Zepto, Blinkit earn ₹15,000–25,000/month with zero income protection. One accident = income wiped.

> **What I built:** A parametric insurance engine that triggers payouts based on GPS work-hour data — no claims, no paperwork, instant payment.

```
Stack  →  FastAPI · React · PostgreSQL · LangChain · Groq · GPS telemetry
Market →  11M+ gig workers · ₹2,400 Cr addressable opportunity
Status →  Prototype stage · Advisor: Japhire Gopi Kannan
```

This is the kind of problem I care about. Real people, real impact, real tech.
